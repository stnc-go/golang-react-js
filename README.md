Running The App
- Backend \
  ``cd cmd/api`` \
  ``go run main.go`` 

- if you want build 
  ``cd cmd/api`` \
  ``go build`` \
  ``./api `` 

- Golang Frontend - run in readinglist root directory \
  ``go build`` \
  ``go run ./cmd/web/*.go `` 

- React + Vite Frontend - run in readinglist-ui directory \
  ``cd frontend/readinglist-ui`` \
  ``npm i`` \
  ``npm run dev``
